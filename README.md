# keycloak-service-social-lark
Keycloak social identity provider for Lark(飞书).


## Run with Jboss
1. 添加jar包到Keycloak服务:  
```
cp target/keycloak-service-social-lark-${version}.jar /opt/keycloak/providers/
```
